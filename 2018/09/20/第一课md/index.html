三、知识点

1.导入模块

import在python中是导入的意思，import后面接我们要导入的模块名字，在import和要导入的模块之间要加空格。例如导入turtle模块：

import turtle

一个模块里面会有很多方法，当我们使用模块里面的方法的时候，需要模块名.方法名，例如调用turtle模块的forward()方法：

turtle.forward(120)

2.召唤海龟

方法：turtle.shape(“turtle”)

turtle是我们这个level使用的模块，前面必须用import turtle导入turtle模块，我们才能使用它。shape(“turtle”)是turtle模块的一个方法，可以在画布中出现一只海龟.

3.画布持续

方法：turtle.done()
当我们第一次调用turtle模块的时候，会生成一张画布。turtle.done()方法可以让它一直持续显示。
注意：如果不用done()方法，那么当程序运行完了之后，画布会自动消失。

4.移动

①turtle.forward(distance)

forward()方法可以让海龟向前移动，括号里是一个参数distance，是一个数值，为海龟移动的距离，以像素为单位。

例如：

turtle.forward(120)

意思就是让海龟向前移动120单位的距离。
②turtle.backward(distance)
backword()方法可以让海龟向后移动，括号里是一个参数distance，是一个数值，为海龟移动的距离，以像素为单位。用法跟forword()方法是一样的。

5.转角

方法：

turtle.left(angle)向左转
turtle.right(angle)向右转

left()、right()方法都是设置海龟转向的方法，left是向左转，right是向右转。括号里的angle是一个参数，意思为角度。数值为需要海龟转向的角度。例如

turtle.left(90)

意思为海龟向左转90度。right方法也是一样的，只不过是向右转。

6.画圆

方法：

turtle.circle(radius)

画圆可以用circle()方法，括号里的radius是一个参数，表示半径，我们可以自己设置数值。例如：

turtle.circle(50)

这行代码意思是画一个半径为50的圆。

7.画笔

方法：

turtle.Pen()

我们用turtle模块来画图，画图肯定要用到画笔，Pen()方法可以为我们提供一支画笔，Pen()的后面不用设置参数。
注意：Pen是一个大写的P，turtle.Pen()方法一般在程序开始的时候就调用。

五、延伸

导入模块有4种方法：

①import module_name

在import后面接我们想导入的模块的名字
当我们使用模块里面的方法或者函数的时候，需要在模块名后面加上方法名。module_name.func().
module_name为我们导入的模块，func为我们想要调用的方法或者函数，我们视频中讲的就是这一种方法。例如：

turtlr.forward()

注意：所有的方法import后面记得要加一个空格，把它和模块名隔开。

②import module_name1,module_name2

导入多个模块，module_name1、module_name2为我们想要导入的第一个模块、第二个模块的名字，一次性可以导入很多个模块，只需要把模块名接在后面就可以了。

例如一次性导入turtle模块、sys模块、os模块

import turtlr,sys,os

from module_name import *

导入module_name里面的所有函数或者方法，*在这里代表所有。这个方法要谨慎使用，如果你导入的模块里面的方法跟其他模块里的方法同名，就会引起冲突
注意：如果导入模块的方法和目前环境引起冲突，可以在导入的时候用as设置别名，例如：

import turtle as t

以后调用turtlr模块就可以用t代替

④from module_name import func

这个方法只导入module_name模块里面的func方法，使用的时候直接func就可以
